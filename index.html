一、Structure and Bonding（結構與鍵結）

1. 共價鍵種類

  • σ 鍵：軌域正面重疊形成，所有單鍵都是 σ 鍵
  • π 鍵：平行 p 軌域側向重疊形成，存在於雙鍵與三鍵中

2. 單鍵、雙鍵、三鍵

  • 單鍵 = 1 個 σ 鍵
  • 雙鍵 = 1 個 σ 鍵 + 1 個 π 鍵
  • 三鍵 = 1 個 σ 鍵 + 2 個 π 鍵

3. 鍵長與鍵強

鍵階越大，鍵越短、越強：
C≡C < C=C < C–C（鍵長）

4. 混成軌域

混成 幾何形狀 鍵角 例子
sp3 四面體 109.5° 烷類
sp2 平面三角形 120° 烯類
sp 直線形 180° 炔類、allene 中央碳

5. 鍵極性

  • C–Br：Cδ+–Brδ−
  • C–N：Cδ+–Nδ−
  • N–H：Hδ+–Nδ−
  • C–F：Cδ+–Fδ−

二、Acids and Bases（酸與鹼）

1. 定義

  • 酸：提供 H+
  • 鹼：接受 H+

2. pKa 觀念

pKa 越小，酸越強

3. 酸性強弱判斷因素

  • 共軛鹼是否穩定
  • 誘導效應
  • 共振效應
  • 電負度
  • 原子大小

4. 例題重點：ethanol vs trifluoroethanol

  • ethanol pKa = 16
  • 2,2,2-trifluoroethanol pKa = 12.4
  • CF3 為強吸電子基，可穩定共軛鹼，因此 trifluoroethanol 酸性較強
  • 酸越強，其共軛鹼越弱

三、Cyclohexane and Chair Conformation（環己烷與椅式構形）

1. 基本概念

  • 每個取代基可位於 axial（軸向）或 equatorial（赤道向）
  • ring flip 會使 axial 和 equatorial 互換

2. 穩定性

大基團通常偏好 equatorial，因為可避免 1,3-diaxial interaction。

3. 重要例子

  • 1-tert-butyl-1-methylcyclohexane 中，tert-butyl 放在 equatorial 最穩定
  • cis-dichlorocyclohexane 某些情況下在單一 chair 中必為一軸一赤道

四、Conformation Analysis（構形分析）

1. Newman Projection

  • 用來觀察單鍵旋轉的不同構形
  • 比較 eclipsed 與 staggered 的能量

2. 能量關係

  • staggered 比 eclipsed 穩定
  • CH3–CH3 eclipsed 排斥最大，能量最高

五、IUPAC Nomenclature（IUPAC 命名）

命名步驟

  1. 找最長主鏈
  2. 主官能基優先
  3. 從最接近雙鍵或取代基的一端編號
  4. 標示取代基位置
  5. 依字母順序排列
  6. 必要時標示 cis/trans 或 E/Z

六、Alkenes and Alkynes（烯類與炔類）

1. 烯類

  • 含有 C=C
  • 碳原子通常為 sp2 混成
  • 常有 cis-trans / E-Z 異構

2. 炔類

  • 含有 C≡C
  • 碳原子通常為 sp 混成
  • 末端炔具酸性，可被強鹼去質子化

3. cis-trans 異構條件

雙鍵兩端的每個碳都必須連接 兩個不同基團，才可能有 cis-trans 異構。

七、Chapter 4：Reactions of Alkenes and Alkynes（烯類與炔類的反應）

1. 加成反應

烯類與炔類最重要的反應為對多重鍵進行加成，使分子變得更飽和。

2. HX 加成

  • 烯類 + HX → 鹵代烷
  • 遵守 Markovnikov's rule
  • H 加到原本氫較多的碳，X 加到較取代的碳

3. X2 加成

  • 烯類 + Br2 / Cl2 → vicinal dihalide
  • 炔類也可與 X2 反應，過量時可得 tetrahalide

4. H2O 加成（水合）

  • 烯類加水生成醇
  • 炔類加水後可經互變異構形成酮或醛

5. H2 加成（氫化）

  • 烯 / 炔可被氫化
  • 常用催化劑：Pd、Pt、Ni

八、Reaction Classification（反應分類）

類型 特徵
Addition 多重鍵打開，加入新原子或基團
Elimination 脫去小分子，形成雙鍵或三鍵
Substitution 一個基團被另一個基團取代
Rearrangement 分子骨架重新排列

九、考前必背重點

  • sp3 = 109.5°；sp2 = 120°；sp = 180°
  • 三鍵最短最強
  • pKa 越小，酸越強
  • 大基團放 equatorial 比較穩定
  • 烯與炔最重要的反應是加成反應
  • HX 加成要會判斷 Markovnikov 規則
  • 雙鍵兩端各有兩種不同基團才可能有 cis-trans 異構
